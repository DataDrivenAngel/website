Anthropic's Fable 5 is Opus on a Good Day

Published 2026-06-10

So far a couple of hours with Antrhopic's new Fable model in Claude Code feels like it consistently does what Opus does on a good day. Claude Code with Fable more consistently has The Magic ™️ that it had in November and December of last year. It does look like it'll be notably more expensive than Opus, and it does seem to run longer without asking for clarification (for better or worse), but the rate at which it just makes software is pretty impressive.
